题目大意：有Q个询问，对每个询问，给出N，M，k，求 1 <= x <= N，1 <= y <= M 中满足 gcd(x, y) = k 的无序对(x, y)的对数。

若先不考虑无序，根据我们前面的经验，构造f(d), F(d):

f(d) = | {(x, y) | gcd(x, y) == d, x <= N, y <= M} |

F(d) = | {(x, y) | (d | gcd(x, y)), x <= N, y <= M} |
